# Stack

## Big-O Notation
`Assumption: We do not change the order of array`
1. Search
    - `Find value, not getting the value at x index`
    - O(n) on average
2. Insert
    - `Pushing the element from the top`
    - O(1) on average
3. Delete
    - `Popping element from the top`
    - O(1) on average
