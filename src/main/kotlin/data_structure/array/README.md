# Array

## Big-O Notation
`Assumption: We do not change the order of array`
1. Search
   - `Find value, not getting the value at x index`
   - O(n) on average
2. Insert
    - `Insert and push back everything comes after`
    - O(n) on average
3. Delete
    - `Delete and push forward everything comes after`
    - O(n) on average
